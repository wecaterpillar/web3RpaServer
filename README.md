web3RPA Server
===============

## 项目介绍

提供web3相关项目的RPA服务，可用于各种社群或者空投活动。

## getting start


## 主要功能

### 租户/会员管理（计划中）
- 租户/会员注册
- 租户全局设置
- 会员用户管理

### 账号/环境管理
- web账号管理
- 指纹环境管理
- 代理配置管理
- 钱包账号管理
- 账号平台列表(查看)
- 浏览插件列表(查看)

### web3项目管理
- 空投项目管理
- 项目账号明细

### RPA流程管理
- rpa流程脚本
- rpa执行计划
- rpa任务详情

### 系统管理
- 操作日志
- 执行终端
